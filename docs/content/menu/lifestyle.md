# 生活習慣（食事・呼吸法・姿勢）

## 1. 食事

### 基本方針
- 糖質：1日120g目安
- 脂質：原則オリーブオイル。トランス脂肪酸を排除。悪玉コレステロールを減らす
- タンパク質：プロテイン、サラダチキン、大豆、チーズ、ナッツなどから
- 避けるもの：缶コーヒー・清涼飲料水・ジュース／添加物／加工肉／空腹時のドカ食い／トランス脂肪酸
- 量：腹八分目

### 朝食
- プロテイン
- ヨーグルト＋シリアル
- パン＋バター／チーズ／ジャム
- コーヒー

### 昼食
- サラダチキン＋サラダ＋BASE Bread のセット
- 食後に少し歩く

### 間食
- チーズ
- 無塩ナッツ
- カカオ70%以上のチョコレート

### 夕食
- 寝る4時間前が理想
- ご飯は少なめで最後
- 食後に体幹トレーニング

### 水分・嗜好品
- 缶コーヒー・清涼飲料水・ジュースは避ける

## 2. 呼吸法

### 目指す呼吸
- 吸う時に胸と腹の両方が膨らむ
- 肩は上がらない
- 吐く時も腹圧を保つ
- 肩甲骨まわりが呼吸で動く感覚

> 呼吸改善は単なるリラックスではなく、**すべてのトレーニングのベース**として位置付ける。

### 練習タイミング
| タイミング | メニュー | 時間 |
| --- | --- | --- |
| 朝 | キャットキャメル中の呼吸意識 |  |
| トレ前 | 新呼吸法（胸＋腹／肩を上げない／肩甲骨を動かす） |  |
| 入浴前ルーチン中 | 各種メニューに呼吸を乗せる |  |

## 3. 姿勢（常時意識）

### ひとことで意識する 5 項目（日常の合言葉）
- 肩甲骨を内側・下に動かす
- 腹式呼吸を意識する
- ガニ股を治す
- 重心が前に行きすぎていないか確認する
- 肩が上がっていないか確認する

### 場面別チェック項目
| 場面 | チェック項目 |
| --- | --- |
| 立位 | 肩甲骨が内側・下にある／重心が前に行きすぎていない／足指に過剰な力が入っていない／肩がすくんでいない |
| 座位（デスクワーク） | 肩甲骨を内側・下に／反り腰になっていない／前重心になっていない |
| 歩行 | ガニ股になっていない／前重心になっていない／母趾球に過剰な負担がかかっていない |

### NG パターン（自分が陥りがちな癖）
- 肩がすくむ
- 反り腰
- 前重心
- ガニ股
- 立っている時に足指に力が入る
- 腰を反らせて胸を張ろうとする

## 4. 睡眠・自律神経・メンタル

### キーワード
- 副交感神経優位の時間を意識的に作る（40代は自律神経のトータルパワーが落ちる前提）
- デフォルトモードを作る
- ワーキングメモリをOFFにする
- 脱マルチタスク思考
- 休養にも集中する
- すべての行動に意味や価値を見出そうとしない
- SNSを見たくなったら危険信号
