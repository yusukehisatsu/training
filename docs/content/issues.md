# 現在の課題と解決の方向性

## 1. 中心仮説

肩・首・腰・ハムストリング・足指の問題は別々ではなく、**呼吸・肋骨・肩甲骨・骨盤・股関節・足部の連動不全**としてつながっている可能性が高い。

単に「筋肉が硬い」のではなく、

- 肩で動作を代償してしまう
- 肩甲骨・胸椎・肋骨・骨盤・腹斜筋群がうまく連動していない
- 呼吸が浅く、肋骨や背中が動きにくい
- デスクワーク姿勢や前重心が影響している
- 股関節・骨盤のズレや体幹の不活性が下半身の張りにつながっている

という構造で起きている。

## 2. 部位別の課題

### 肩・首・肩甲骨まわり
- 現状：肩こり、肩甲骨まわりの硬さ／使いにくさ。僧帽筋上部が硬くなりやすい。
- 想定される原因：肩で動作を代償してしまう癖。胸椎・肋骨・肩甲骨が連動して動かず、肩をすくめる方向で動いてしまう。「僧帽筋上部を揉む」だけでは根本解決にならない。
- 目指す状態：
  - 肩甲骨が内側かつ下に収まっている状態
  - 腰を反らなくても胸が張れている状態（**胸骨から反れる**ようになる）
  - 肩が上がらず、鎖骨まわりに力が入っていない状態
  - 肩甲骨を**下制・後傾・外旋**できる
  - 胸骨・肋骨・胸椎から動ける
  - 立ち姿の参考：僧帽筋可動の W 字トレーニングをしてそのまま手を下げた状態

### 腰・骨盤・体幹
- 現状：腰まわりの不安定さ。反り腰や骨盤のズレ。腹斜筋群の不活性。
- 想定される原因：腰を反らせて胸を張ろうとしてしまう。肋骨・胸椎・肩甲骨を使えていない。腹斜筋群が眠っている。
- 目指す状態：腰を反らせるのではなく、**肋骨・胸椎・肩甲骨を使って上半身を整える**。腹斜筋群を活性化して骨盤を安定させる。サイドプランク系・サイドプランクローリングなどで体幹を活性。

### 下半身（右ハムストリング・臀筋・股関節）
- 現状：右ハムストリングの張り。毎日ストレッチしても取れにくい。
- 想定される原因（複合）：
  - 骨盤の左右差
  - 腹斜筋群の不活性
  - 中臀筋・大臀筋の使いにくさ
  - 股関節の安定性不足
  - 前重心やガニ股傾向（**ガニ股は中臀筋が硬くなる原因にもなり得る**）
- 目指す状態：伸ばすだけでなく、骨盤を安定させて中臀筋・内転筋・腹斜筋を使えるようにする。サイドプランク／クラム系／ヒップアブダクション／内転筋活性／デッドバグ／骨盤を安定させた片脚系動作で土台を作る。

### 呼吸
- 現状：基本的に呼吸が浅い。
- 想定される原因：肋骨・胸椎・背中が動きにくく、結果として肩甲骨も動きにくい。肩甲骨は肋骨の上を滑るため、肋骨まわりが固いと肩甲骨も動かない。
- 目指す状態：
  - 吸う時に胸と腹の両方が膨らむ
  - 肩は上がらない
  - 吐く時も腹部の圧を保つ
  - 肩甲骨まわりが呼吸で動く感覚

呼吸改善は単なるリラックスではなく、**すべてのトレーニングのベース**として位置づける。

## 3. 改善の優先順位

1. 呼吸を深くする
2. 肩をすくめず肩甲骨を下げる
3. 腰を反らずに胸を張る
4. 腹斜筋・体幹を活性化する
5. 股関節と臀筋を使えるようにする
6. 足指に過剰な負担をかけない靴・歩き方にする
7. 筋トレは重量よりフォームと連動性を優先する

## 4. 活性すべき筋／抑制すべき筋

> トレーナーフィードバックを反映して更新する。
> 同じ筋が「硬くてほぐすべき」と「活性すべき」両方に該当する場合があることに注意。

### 活性すべき筋
| 筋肉 | 役割／なぜ活性が必要か | アプローチ種目 |
| --- | --- | --- |
| 肩甲下筋 | ローテーターカフの一つ（肩関節内旋・肩の安定）。肩で代償しないために肩関節の深層安定が必要 | 肩インナーマッスル3セット（仰向け脇締め内側引き）、肩甲骨スタビリティ |
| 棘上筋 | ローテーターカフの一つ（肩関節外転の初動・肩関節の動的安定）。硬くなりやすく抑制も必要だが、肩甲上腕リズムの土台として活性も必要な「両方に該当」する筋 | 肩インナーマッスル（軽負荷の外転系）、肩甲骨スタビリティ |
| 広背筋 | 引く動作の主役。肩甲骨を下制しながら腕を引き寄せる。肩のすくみを抑制する | ラットプルダウン、シーテッドロー、チューブローイング、腕を引くやつ |
| 僧帽筋下部 | 肩甲骨の下制・後傾。「肩甲骨を内側かつ下に収める」ために最重要 | チューブYロー、僧帽筋トレ（うつ伏せ反り）、ウォールスライド、壁肩甲骨スクワット |
| 上腕三頭筋 | 肘の伸展。プッシュ動作の安定。肩甲骨と連動する押す力 | 上腕三頭筋トレ、チェストプレス、ショルダープレス、プッシュアップ |
| 腹横筋 | 最深部のインナーコア。腹圧で骨盤と腰椎を安定させる。呼吸の土台 | デッドバグ、新呼吸法、腹式呼吸ストレッチ |
| 内腹斜筋 | 骨盤の安定、回旋制御。右ハム張りの原因として「腹斜筋群の不活性」が指摘されている | サイドプランク・ウィンドミル、サイドプランクローリング、腹筋と胸筋連動 |

### 抑制すべき筋
| 筋肉 | なぜ抑制（過緊張をほぐす）が必要か | アプローチ |
| --- | --- | --- |
| 棘上筋 | 過緊張で肩のすくみと肩関節インピンジメントの原因になる | ボールリリース（風呂後）、肩肩甲骨ケア |
| 僧帽筋上部 | 肩こりの主犯。肩甲骨を上に引き上げてしまい、僧帽筋下部の活性を妨げる | 風呂中にほぐし、風呂後にローラー／ボールリリース、マッサージガン |
| 小胸筋 | 肩甲骨を前傾させ巻き肩の原因に。肩甲骨を後傾させるために抑制が必要 | 風呂中にほぐし、大胸筋・鎖骨下筋リリース、胸伸ばしストレッチ |
| 大胸筋 | 過緊張で巻き肩を促進し、肩甲骨後傾を妨げる | 風呂中にほぐし、胸伸ばしストレッチ、肩伸ばし |
| 上腕二頭筋 | 過緊張で肩前部を引き込み、巻き肩を悪化させる | ローラーリリース、ストレッチ |
| 前腕屈筋群 | デスクワーク・スマホで慢性的に張りやすい。前腕の硬さは肩のすくみまで波及 | ストレッチ、マッサージガン |
| 腰方形筋 | 反り腰・骨盤の左右差の原因に。右ハム張りの背景である骨盤のズレと関連 | 腰横ストレッチ、ボールリリース |
