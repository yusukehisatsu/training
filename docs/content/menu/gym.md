# ジムメニュー

利用ジム：Blue Fitness 24 勝どき店（MATRIX／SPIRIT／DRAX／ZERO-i 等を備えるジム）

## 1. 週間サイクル

> **前提**：朝イチ出勤前のジム、滞在25分。基本的に毎日通う。
> **重視ポイント**：①背中（広背筋・僧帽筋下部） ②脚（デッドリフト中心、ハム・大臀筋・中臀筋） ③有酸素1日。
> **除外方針**：抑制すべき筋（大胸筋・小胸筋・三角筋・上腕二頭筋・棘上筋）を直接狙う種目はメニューに組み込まない。押す日（胸・肩）は設けず、活性すべき筋のうち上腕三頭筋・肩甲下筋は別日で補う。

| 曜日 | 内容 | メイン狙い |
| --- | --- | --- |
| 月 | **引く日 A（広背筋・厚み）** ★ | シーテッドロー中心 |
| 火 | **脚 A（デッドリフト＋ハム）** ★ | ルーマニアンデッドリフト |
| 水 | **有酸素＋体幹コア** | リカンベントバイク＋ベアクロール |
| 木 | **引く日 B（広背筋・幅）** ★ | ラットプルダウン中心 |
| 金 | **脚 B（デッドリフト＋臀部）** ★ | デッドリフト＋中臀筋／大臀筋 |
| 土 | **休み（完全オフ）** | 入浴ケア・寝る前ストレッチに時間を使う |
| 日 | 上腕三頭筋 ＋ 肩インナー | 活性すべき筋（上腕三頭筋・肩甲下筋）をカバー |

### サイクル設計の根拠
- **背中×2、脚×2** で重視ポイント①②に厚みを持たせる。A/B 分割で同部位でも狙う筋繊維（厚み／幅、ハム／中臀筋）を変える
- 抑制すべき筋（大胸筋・三角筋・上腕二頭筋）を直接刺激する種目（マルチプレス・ショルダープレス・ペックフライ・バイセプスカール・リアデルト等）は全てカット
- 活性すべき筋のうち、引く日と脚日でカバーされない**上腕三頭筋・肩甲下筋**は日曜に独立して配置
- 水曜は「有酸素」と「体幹コア」を合体。**リカンベントバイク**で下半身の血流促進＋腰サポート、**ベアクロール**で腹横筋・前鋸筋・肩甲骨安定筋群を刺激。両方とも母趾への衝撃ゼロ
- **土曜を週1の完全オフ日**にして回復に充てる。入浴ケア・寝る前ストレッチを丁寧に
- **右ハム張り対策の核**である腹斜筋群活性化は、ウォーミングアップ（デッドバグ＋サイドプランクローリング）で毎日担保

## 2. 1回のセッション基本サイクル（25分構成）

| ブロック | 時間 | 内容 |
| --- | --- | --- |
| ウォーミングアップ | 3-5分 | 呼吸＋体幹活性＋肩甲骨／前鋸筋を起こす |
| メイン種目 2-3個 | 18-20分 | 各3-4セット、種目間レストは短め（45-60秒） |
| クールダウン | 1-2分 | 軽いストレッチ／呼吸を戻す |

> 25分の制約上、1部位に集中して刺激の質を高める方針。複数部位を1日に詰め込まない。

## 3. ウォーミングアップ（毎回 3-5分）

> 目的：呼吸を整え、肩甲骨を下制／後傾／外旋できる状態にし、体幹を活性化してから本番に入る。
> 背中・脚いずれの日でも、腹筋群（特に内腹斜筋・腹横筋）を活性してから入らないと、肩甲骨が挙上したり骨盤が不安定なまま動いてしまう。

- [ ] 呼吸の確認 30秒（胸＋腹が膨らむ、肩が上がらない）
- [ ] キャットキャメル 10回（胸椎可動＋腹筋群活性）
- [ ] デッドバグ 10回（腹横筋・体幹深層を活性／背中をマットから離さない・腹圧を抜かない）
- [ ] サイドプランクローリング 各 10回（内腹斜筋・腹斜筋を活性／肩甲骨を下げたまま回旋）
- [ ] 前鋸筋プッシュアップ 10回（肩甲骨を動かせる状態を作る）

## 4. トレーニング中の共通注意点

- [ ] 肩をすくめない／肩甲骨を**下制・後傾・外旋**した状態でフォームを作る
- [ ] 腰を反らずに胸を張る（肋骨・胸椎で動く）
- [ ] 重量よりフォームと連動性を優先
- [ ] 背中トレ前は腹筋群を必ず活性してから入る
- [ ] 右ハムの張り対策には腹斜筋群の活性化が重要
- [ ] デッドリフトは**肩甲骨を下げた位置・骨盤後傾・つま先まっすぐ**を維持する

### 避ける／非推奨種目
| 種目 | 理由 | 代替 |
| --- | --- | --- |
| ロータリートルソー | 腹筋よりも脊柱への負担が高い | サイドプランクローリング（同等の効果＋機能的） |

### 抑制すべき筋を狙う種目はメニューから外す
| 抑制すべき筋 | 該当する代表種目（このメニューでは組まない） |
| --- | --- |
| 大胸筋・小胸筋 | チェストプレス、マルチプレス、ペックフライ |
| 三角筋（前部・中部・後部） | ショルダープレス、サイドレイズ、リアデルト |
| 棘上筋 | サイドレイズ |
| 上腕二頭筋 | バイセプスカール、プリーチャーカール |
| 腰方形筋 | 過度な側屈・回旋系（過負荷の体側種目） |

### 土：休み（完全オフ）
> 週1日の完全休養日。ジムには行かない。入浴中ほぐし・入浴後ローラー・寝る前ストレッチを丁寧に行い、回復に充てる。
