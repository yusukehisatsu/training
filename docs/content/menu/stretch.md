# ストレッチメニュー

> 役割：入浴動線に沿った**リリース**（section 1：入浴中／section 2：入浴後）と、**寝る前の下半身伸長**（section 3）に特化する。
> ほぐし対象筋は短く・硬くなっている可能性が高いため、「抑制すべき筋」とされたものを優先する。
> 風呂前 5 分ルーチン（活性＋モビリティ）に含まれる種目（オープンブック／ダウンドッグ／キャットキャメル／壁背骨／壁肩甲骨スクワット／腹式呼吸ストレッチ／胸伸ばし）は、原則ここでは重複させない。

## 1. 入浴中（マッサージ・ほぐし）

| 部位 | 道具 | 狙い | 意識ポイント |
| --- | --- | --- | --- |
| 側頭筋・咬筋 | 指 | 噛みしめ・首肩のテンション緩和 | リリースする側に呼吸を送る |
| 大円筋 | 指 | 肩甲骨可動の改善、肩のすくみ抑制 | 肩をすくめない |
| 胸鎖乳突筋 | 指 | 首前面のリリース、呼吸の深さに影響 | 強く押し込みすぎない／呼吸を止めない |
| 大胸筋・鎖骨下筋 | 指 | 巻き肩リリース、肩甲骨を後傾させやすくする | 鎖骨下を内→外に滑らせる／肩は脱力 |
| 小胸筋 | 指 | 巻き肩リリース、肩甲骨前傾を解く（抑制すべき筋） | 鎖骨外側〜烏口突起付近を狙う／力みすぎない |
| 上腕二頭筋 | 指 | 肩前部の引き込みを抑え、巻き肩悪化を防ぐ（抑制すべき筋） | 二頭筋の腱付着部を避け、筋腹をほぐす |
| 僧帽筋上部 | 指 | 肩こり緩和（抑制すべき筋） | 肩をすくめずに緩める |
| 肩甲下筋 | 指 | 肩関節の内旋筋、肩の動きの改善 | 脇下から肩甲骨内側に指を入れる |

## 2. 入浴後（フォームローラー／ボール）

| 部位 | 道具 | 狙い | 意識ポイント |
| --- | --- | --- | --- |
| ヒラメ筋 | ローラー／ボール | 足首の可動、左母趾痛対策 | アキレス腱付着部より上をほぐす |
| 腓腹筋 | ローラー | ふくらはぎの硬さ緩和 | 内側／外側両方を当てる |
| 大腿二頭筋 | ローラー | 右ハム張り対策（ほぐしのみで完結しない／核は腹斜筋活性） | 右側を丁寧に／反対側と比較 |
| 半腱様筋 | ローラー | 内側ハム | 膝裏付着部直上を避ける |
| 前脛骨筋 | ローラー／ボール | 足首の可動、母趾痛対策 | 脛骨外側のラインに沿って |
| 上腕三頭筋 | ローラー | 肩甲骨可動 | 肘関節直上を避け、筋腹を当てる |
| 大円筋 | ローラー／ボール | 肩甲骨可動 | 脇下後面の硬い箇所をピンポイントで |
| 広背筋上部 | ローラー | 肩甲骨の下制を妨げる硬さの緩和 | 肋骨側面に沿って |
| 棘上筋 | ボール／ガン | 肩のすくみ・インピンジメント抑制（抑制すべき筋） | 肩甲棘の上のくぼみにボールを当てる／肩は下げる |
| 棘下筋 | ボール／ガン | 肩関節後面のリリース、肩甲骨可動の改善 | 肩甲骨棘下窩を狙う／肩は下げる |
| 前鋸筋 | ボール | 肩甲骨を動かせる状態を作る | 肋骨外側に沿って当てる |
| 大殿筋 | ローラー／ボール | 股関節の安定、ハム張り対策 | 仙骨直上ではなく筋腹に当てる |
| 中殿筋 | ボール | 骨盤の安定、ハム張り対策 | 腸骨稜直下〜大転子の間 |
| 腰方形筋 | ボール | 反り腰・骨盤左右差・右ハム張りの背景緩和（抑制すべき筋） | 肋骨下端と腸骨稜の間／左右差を確認 |
| 前腕屈筋群 | ローラー／指 | デスクワーク由来の張り緩和、肩のすくみへの波及を断つ（抑制すべき筋） | 手首付近の腱より、筋腹を狙う |
| 菱形筋 | ボール | 肩甲骨周辺 | 肩甲骨内側縁に沿って |
| 僧帽筋上部 | ボール／ガン | 肩こり緩和（抑制すべき筋） | 首付け根に押し込みすぎない |
| 内転筋 | ローラー | 内転筋群のリリース | うつ伏せで内股側を当てる／鼠径部・膝裏直上を避ける |

## 3. 寝る前ストレッチ

| メニュー | 狙う筋肉 | 意識ポイント |
| --- | --- | --- |
| ハムストリングストレッチ | ハムストリング（特に大腿二頭筋・右側） | 右側はとくに丁寧に／骨盤を立てたまま前屈 |
| 鳩のポーズ | 大殿筋・梨状筋 | 股関節の柔軟性／前足側のお尻が伸びる感覚／骨盤を真っ直ぐに |
| クアッドストレッチ | 大腿四頭筋 | 反り腰にならないように／骨盤後傾を保つ |
| 腰横ストレッチ | 腰方形筋 | 左右各／骨盤は起こしたまま（ボールが入っているイメージ）横に倒れる／反り腰にならない |

## 4. 共通注意点

- 肩甲骨を下制した状態を維持する／肩をすくめない（特にほぐし中も肩を上げない）
- 腰を反らない（特に section 3 の前屈・四頭筋ストレッチ）
- 呼吸を止めない（吸うときに胸と腹の両方が膨らむ感覚を保つ）
- リリース中は痛みを我慢しすぎない（過剰刺激は逆に防御性緊張を高める）
